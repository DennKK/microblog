# Microblog

Microblog is a Java-based web application created for educational purposes. This application allows users to register/login and view and create posts.

## Technologies Used

- Java
- Spring Framework (Spring boot, MVC, Security)
- Maven
- PostgreSQL
- Thymeleaf
